# 网页卡牌参考资料

这里保存游戏局外无法预览内容的网页证据，例如普通闪（中立闪/通用闪）、专属灵光一闪与神闪的组合画面。

这些文件与 `inbox`、`review` 中的实机样本严格分开：

- 网页图片不是训练样本，也不能用于统计游戏内分辨率、裁切位置或颜色阈值。
- 所有结论保持 `pending`，必须由当前版本实机画面复核后才能写入 `data/cards/characters`。
- 每张图片记录来源网址、原图网址、抓取日期、字节数和 SHA-256，避免来源或文件被悄悄替换。
- 只整理角色牌的识别和效果证据；中立牌、怪物牌即使出现在参考图中也不建立卡牌定义。
- 网页攻略可能落后于版本。冲突信息必须并列保存，不能自动选择看起来更合理的一项。

`haide_mali/flash_layers.pending.json` 是当前试点资料。其主要用途是：

1. 规定普通闪与专属灵光一闪互斥、神闪可额外叠加的识别标签结构。
2. 为海德玛丽四张可闪角色牌保存候选普通闪词条。
3. 保存旧通用神闪池、新物主专属神闪池以及版本差异。
4. 告诉后续采集器“还缺什么”，而不是直接把网页文字提升为正式卡牌事实。
